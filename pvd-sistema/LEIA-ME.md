# 🍔 Lanche PDV — Pacote de Deploy

## 🚀 Instalação em 1 comando

```bash
chmod +x instalar.sh
./instalar.sh
```

O script pergunta:
- Qual modo (domínio com HTTPS OU apenas IP)
- Se for domínio: qual o domínio + email

E faz **tudo o resto automaticamente** (Docker, .env, SSL, migrations, seed).
